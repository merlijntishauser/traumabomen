Action accent

Interactive color. Green in dark theme, indigo in light. Used for primary buttons, links, focused fields, focus rings, segmented-toggle active state, the SELECTED ring on a node. Brand --color-accent (logo, leaf, tree edges) stays green in both themes.

Dark theme — green
--color-action
#2d8a5e · forest green
Light theme — indigo
--color-action
#4f46e5 · indigo
Rationale: cream + indigo reads editorial / "reflection tool" — fits the brand voice better than green-on-cream, which felt generic-wellness. The forest stays the forest; only the *actions* shift register.
